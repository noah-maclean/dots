Arch Linux dotfiles with chezmoi

## Installation

### Laptop

1) `chezmoi init git@github.com:noah-maclean/dots.git`
2) run scripts/install.sh to install packages


### Desktop

`chezmoi init git@github.com:noah-maclean/dots.git --branch desktop`
